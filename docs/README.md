# 3주 차 프리코스 미션 - 로또

📃해당 프로젝트는 간단한 로또 게임을 구현한 프로젝트입니다.
<br>

# 📬입력 예시

## **숫자가 아닌 입력 시 공통 예외**

- [wrong]`~ 입력해 주세요.`

```text
일,이,삼,사,오,육 or 문자열
[ERROR] 숫자만 입력 가능합니다.
```

## **로또 구입 금액을 입력받는다.**

    - 구입 금액은 `1,000원 단위`로 입력받는다.
    - 최대 구입 금액은 `100,000`원이다.
    - 최소 구입 금액은 `1,000`원이다.

### **구입 금액 입력 예시**

- [right]`구입금액을 입력해 주세요.`

```text
8000
```

```text
10000
```

- [wrong]`구입금액을 입력해 주세요.`

```text
500
[ERROR] 로또 구입 금액은 최소 1,000원입니다. 
```

```text
110000
[ERROR] 최대 구입 가능 금액은 100,000원입니다. 
```

```text
5555
[ERROR] 1,000원 단위로 입력해 주세요.
```

## **당첨 번호를 입력받는다.**

    - `6개의 숫자`가 입력되어야 한다.
    - `쉼표(,) 기준`으로 구분한다.
    - `중복되지 않아야` 한다.
    - `1부터 45 사이의 숫자`여야 한다.

### **당첨 번호 입력 예시**

- [right]`당첨 번호를 입력해 주세요.`

```text
1,2,3,4,5,6
```

```text
45,43,42,41,40,39
```

- [wrong]`당첨 번호를 입력해 주세요.`

```text
1,2,3,4,5 or 1,2,3,4,5,6,7
[ERROR] 로또 번호는 6개여야 합니다.
```

```text
1 2 3 4 5 6 or 123456
[ERROR] 쉼표(,)로 구분해 주세요.
```

```text
1,1,2,3,4,5
[ERROR] 로또 번호는 중복될 수 없습니다.
```

```text
1,2,3,4,5,46 or 0,1,2,3,4,5
[ERROR] 로또 번호는 1부터 45 사이의 숫자여야 합니다.
```

## **보너스 번호를 입력받는다.**

    - `당첨 번호와 중복되지 않아야` 한다.
    - `1부터 45 사이의 숫자`여야 한다.

### **보너스 번호 입력 예시**

`ex) 당첨 번호 : [1, 2, 3, 4, 5, 6]`

- [right]`보너스 번호를 입력해 주세요.`

```text
7
```

- [wrong]`보너스 번호를 입력해 주세요.`

```text
6
[ERROR] 보너스 번호는 당첨 번호와 중복될 수 없습니다.
```

```text
50
[ERROR] 로또 번호는 1부터 45 사이의 숫자여야 합니다.
```
